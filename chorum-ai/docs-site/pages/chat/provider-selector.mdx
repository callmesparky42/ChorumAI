## The Options

### Auto (Recommended)

Let Chorum decide. The router picks based on:
- Task type detected from your message
- Available budget
- Provider capabilities
- Cost optimization

For most people, this is the right choice.

### Specific Provider

Force a specific provider:
- **Claude Sonnet** â€” Fast, capable, good default
- **Claude Opus** â€” Best reasoning, higher cost
- **GPT-4 Turbo** â€” Strong all-rounder
- **GPT-4o** â€” Faster GPT-4
- **Gemini 1.5 Pro** â€” Long context, multimodal
- **Ollama** â€” Local, free, private
- **LM Studio** â€” Local, free, private

### Why Force a Provider?

| Reason | Pick |
|--------|------|
| Need best quality | Claude Opus, GPT-4 |
| Want consistent voice | Stick to one provider |
| Processing huge context | Gemini 1.5 Pro |
| Privacy-sensitive task | Ollama (local) |
| Out of API credits | Local models |
| Comparing outputs | Force different ones |

---

## Selection Persists

Your selection persists for the current conversation. If you pick Claude Opus, all messages in this conversation use Claude Opus.

Start a new conversation to reset to Auto.

---

## Visual Indicators

The provider pill changes color based on selection:

| State | Appearance |
|-------|------------|
| Auto mode | Gray/neutral |
| Claude | Orange |
| OpenAI | Green |
| Google | Blue |
| Local | Purple |

This helps you remember what you're using.
